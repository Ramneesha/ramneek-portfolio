# AWS S3 Static Portfolio Hosting

## Project Overview
This project demonstrates hosting a static portfolio website using AWS S3 and CloudFront.

## Technologies Used
- HTML
- CSS
- AWS S3
- AWS CloudFront

## Steps Performed
1. Created static website using HTML and CSS
2. Uploaded files to S3 bucket
3. Enabled static website hosting
4. Configured bucket policy for public access
5. Generated S3 website endpoint
6. (In progress) CloudFront setup for HTTPS
